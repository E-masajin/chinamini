# いつでもクイズ - サービス企画書

## 📌 企画の背景と動機

### なぜこのサービスを作ったのか

企業イベントや交流会において、参加者同士の交流を深める手段として「クイズ大会」は古くから活用されてきました。しかし、従来の早押しクイズや対面形式のクイズには、いくつかの課題がありました。

**私たちは、もっと公平で、もっと柔軟で、もっと多様な交流を生み出せるクイズプラットフォームを作りたいと考えました。**

---

## 🎯 やりたいこと・面白そうだと思ったこと

### 1. 「第三の交流ツール」としての可能性

現在、企業間や社内での交流手段は主に以下の2つに限られています：

- **名刺交換会**: 形式的、受動的
- **懇親会・飲み会**: カジュアル、時間・場所の制約あり

私たちが作りたかったのは、**これらとは異なる「第三の交流手段」**です。

**クイズというゲーミフィケーションを通じて：**
- 自然な会話のきっかけが生まれる
- 競争と協力が混在する独特の一体感
- オンライン・オフライン問わず実施可能
- 準備が簡単で、いつでもどこでも開催できる

### 2. 多様な対戦形式で、様々なシーンに対応

**こんなシーンを想像しました：**

#### 個人戦
- 社内の知識コンテスト
- 研修後の理解度チェック
- カジュアルな腕試し

#### チーム戦
- 部署対抗戦で部署の結束力向上
- プロジェクトチーム同士の交流
- 新入社員とベテランの混成チーム戦

#### 企業戦
- 取引先企業との交流イベント
- 業界団体での知識コンテスト
- 複数企業が参加する展示会・カンファレンスでの企画

**これまでにない「企業同士が対等に競い合う場」が生まれることに、大きな可能性を感じました。**

### 3. 時間と場所の制約を超える

早押しクイズや対面クイズの最大の課題は、**全員が同じ時間・同じ場所に集まる必要がある**ことでした。

**私たちのアプローチ：**
- 期間内であれば好きな時間に参加可能
- リモートワーク環境でも実施可能
- 時差がある海外拠点とも一緒に参加できる

**「いつでも、どこでも参加できるクイズ大会」という新しい体験を提供したかったのです。**

## 🚫 従来のアイデアの課題と、私たちの解決策

### 課題1: 早押しクイズの「反射神経ゲーム化」問題

**従来の課題：**
- 早押しクイズは反応速度が重視され、知識よりも「ボタンを押す速さ」で勝敗が決まる
- ネットワーク遅延により、リモート参加者が圧倒的に不利
- 落ち着いて考える時間がなく、知識を問う本来の目的が薄れる

**私たちの解決策：**
```
✅ 「期間限定・1回のみ回答」方式を採用
✅ 全員が同じ問題数をじっくり考えて回答
✅ スコアと回答時間の両方を評価指標に
```

**結果：** 知識と思考力を公平に評価できる仕組みに。

---

### 課題2: カンニング・情報漏洩問題

**従来の課題：**
- 同じ問題を全員に出題すると、先に受けた人が後の人に情報を漏らせる
- 監視が難しく、不正を防ぐ方法がない
- 公平性が保てず、参加者のモチベーション低下

**私たちの解決策：**
```
✅ ユーザーIDから問題群を自動振り分け
✅ 各ユーザーは異なる問題セットを受け取る
✅ 10グループ × 各5問 = 50問プールから自動選択
```

**アイデアの核心：**
IDの末尾1桁（0-9）から問題群を決定することで、
- カンニングが物理的に不可能に
- 管理者の手間なく自動で公平性を担保
- 全員が「自分だけの問題」に挑戦する特別感

**結果：** 事前情報が一切役に立たない、完全公平なクイズ環境を実現。

---

### 課題3: 個人戦のみでは限界がある

**従来の課題：**
- 個人戦だけでは、参加者同士の交流が生まれにくい
- 成績優秀者だけが目立ち、他の参加者は疎外感を感じる
- 企業間イベントでは「会社の代表」として参加する意識が薄い

**私たちの解決策：**
```
✅ チーム戦・企業戦モードを追加
✅ 平均正答率と平均回答時間で集計
✅ 最低参加人数を設定可能
```

**なぜ平均値なのか：**
- 合計スコアだと参加人数が多いチームが有利
- 平均値なら少数精鋭チームも大人数チームも公平
- 「チーム全員が頑張る」文化が生まれる

**結果：** 個人の成績も大事だが、チーム全体で協力する雰囲気が醸成される。

---

### 課題4: イベント準備・運営の手間

**従来の課題：**
- 問題作成、参加者管理、集計作業がすべて手作業
- リアルタイムで集計できず、結果発表まで時間がかかる
- 運営側の負担が大きく、気軽に開催できない

**私たちの解決策：**
```
✅ 管理者画面で全てを一元管理
✅ イベント作成、問題登録、参加者管理が簡単
✅ ランキングは自動集計・リアルタイム表示
✅ 一度問題を登録すれば、何度でも再利用可能
```

**結果：** 運営負担が大幅に軽減され、「思い立ったらすぐ開催」が可能に。

## 💡 あえてこの仕組みにした理由

### 1. 「1回のみ回答」という制約

**あえて制約を設けた理由：**
- 緊張感と真剣さが生まれる
- 一発勝負の面白さ
- リトライ不可により、公平性が保たれる

**考え方：**
人生の重要な試験（入試、資格試験）は1回勝負です。その緊張感と達成感を、カジュアルなクイズでも体験してほしいと考えました。

---

### 2. ID末尾から問題群を決定する独自ロジック

**なぜこの方式を選んだのか：**

❌ **採用しなかった方法：**
- ランダム選択 → 同じユーザーが再挑戦時に問題が変わってしまう
- 事前に問題群を割り当て → 管理者の手間が増える
- 全員同じ問題 → カンニング可能

✅ **私たちの方式：**
- IDは変わらないため、再参加しても同じ問題群
- 自動計算なので管理不要
- 予測不可能性が高く、カンニング不可

**この仕組みの美しさ：**
シンプルなルール（末尾1桁）で、公平性・再現性・管理容易性をすべて実現。

---

### 3. 回答時間を評価に含める理由

**なぜスコアだけではダメなのか：**
- 同点が多発する可能性
- 知識の速度も重要なスキル
- 「早く・正確に」という実務で求められる能力を評価

**ただし、早押しクイズとは違う：**
- 回答時間は第二優先（第一はスコア）
- じっくり考える時間は十分に確保
- 知識が最優先、速度は同点時のみ影響

---

### 4. 平均値でチーム/企業を評価する理由

**合計値ではダメなのか？**

**問題点：**
- 大企業が人数で圧倒してしまう
- 少数精鋭チームが不利
- 「数の暴力」になってしまう

**平均値のメリット：**
- 少人数でも戦える
- 各メンバーの責任感が高まる
- 「全員が主役」という意識が生まれる

**最低参加人数の設定：**
1人だけのチームでは意味がないため、最低人数（例: 3人以上）を設定可能に。

## 🌟 このサービスが生み出す新しい価値

### 1. 新しい交流の形

従来の名刺交換や飲み会とは異なる、**「競争と協力が共存する交流空間」**を提供します。

- クイズ終了後の「あの問題難しかったね」という会話
- チーム内での「次はもっと頑張ろう」という団結
- 企業間での健全な競争意識

### 2. 時間と場所の制約を超えた参加

リモートワーク時代に最適な、**「非同期型イベント」**の新しい可能性。

- 世界中のどこからでも参加可能
- 時差があっても問題なし
- 仕事の合間に気軽に参加

### 3. データに基づく知見の蓄積

管理者画面で参加者データを分析することで：
- 問題の難易度調整
- 参加者の知識傾向把握
- イベント改善のためのPDCAサイクル

## 🎨 今後の拡張可能性

### 現在実装済みの機能でも、こんな使い方ができる

1. **研修効果測定**
   - 研修前後でクイズを実施し、理解度向上を測定
   
2. **採用イベント**
   - 企業説明会の後にクイズ実施
   - 優秀者を特別選考に招待

3. **業界団体のイベント**
   - 展示会場でQRコードを配布
   - 期間中に参加してランキング上位者に景品

### 将来追加したい機能（アイデア段階）

1. **Googleマップ連動クイズ**
   - 地域や都市を当てるゲーム
   - 営業マンの地理知識コンテストに最適

2. **日替わり問題配信**
   - 毎日1問ずつ配信
   - 継続的なエンゲージメント維持

3. **カスタムブランディング**
   - 企業ロゴやテーマカラーを設定
   - よりブランド体験に近いクイズ環境

## 🎯 まとめ：このサービスが目指すもの

私たちは、**「いつでも、どこでも、誰とでも楽しめる公平なクイズプラットフォーム」**を作りました。

### 核心的な価値

1. **公平性**: カンニング不可能な独自の問題振り分けシステム
2. **柔軟性**: 個人戦・チーム戦・企業戦の3モード対応
3. **簡便性**: 管理者画面で運営が簡単
4. **拡張性**: 様々なシーンに応用可能

### このサービスが実現すること

- 従来の交流イベントにはない「第三の交流手段」の提供
- 時間と場所の制約を超えた、新しいイベント体験
- 公平で、楽しく、意味のある競争の場

---

**「クイズ」という古くからある娯楽を、現代のテクノロジーで再発明する。**

**それが、私たちの「いつでもクイズ」です。**
